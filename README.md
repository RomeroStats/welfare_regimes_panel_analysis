# welfare_regimes_panel_analysis
Este proyecto investiga los determinantes del índice de Gini en distintos países utilizando modelos econométricos para datos panel. Se analiza cómo variables macroeconómicas (PIB, Inversión Extranjera) y de gasto social (Educación, Salud, Seguridad Social) impactan la desigualdad a lo largo del tiempo.
